# ToDo Application

This is a simple ToDo application built with JavaScript, HTML, and CSS. It allows users to manage personal and professional activities by adding, removing, and displaying tasks.

## Project Structure

```
todo-app
├── index.html        # Main HTML interface
├── scripts           # JavaScript files
│   └── app.js       # Main application logic
├── styles            # CSS files
│   └── style.css     # Styles for the application
├── package.json      # NPM configuration file
└── README.md         # Project documentation
```

## Features

- Add new tasks
- Remove existing tasks
- Display all tasks
- Simple and clean user interface

## Installation

1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd todo-app
   ```
3. Install dependencies:
   ```
   npm install
   ```

## Usage

1. Open `index.html` in your web browser.
2. Use the input field to add new tasks.
3. Click on the task to remove it from the list.
